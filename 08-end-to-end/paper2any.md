# Paper2Any

> 学术论文 → draw.io 图 / PPT / 海报 / 视频

| 属性 | 值 |
|------|-----|
| **仓库** | https://github.com/OpenDCAI/Paper2Any |
| **类型** | 端到端转换管线 |
| **部署** | Docker |

## 核心流程

1. 输入 PDF 论文
2. LLM 理解论文内容（方法 / 架构 / 实验结果）
3. 自动生成：
   - **draw.io** 架构图 (.drawio)
   - **PPT** 演示文稿
   - **海报** (学术会议格式)
   - **视频** 讲解（TTS + 动画）

## Agent 场景

Agent 驱动全流程：论文解析 → 结构化提取 → 多格式产出。适合科研汇报、论文答辩准备。
