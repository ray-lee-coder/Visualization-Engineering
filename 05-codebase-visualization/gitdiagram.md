# GitDiagram

> 任意 GitHub 仓库 → 交互式架构图

| 属性 | 值 |
|------|-----|
| **仓库** | https://github.com/ahmedkhaleel2004/gitdiagram |
| **类型** | Web 应用 |
| **部署** | gitdiagram.com |

## 核心用法

将 GitHub URL 中的 `hub` 替换为 `diagram`：

```
https://github.com/user/repo   →   https://gitdiagram.com/user/repo
```

页面自动生成仓库的交互式架构图，支持：
- Zoom / Pan
- 点击文件/目录跳转
- 依赖关系可视化

## Agent 场景

快速理解开源项目结构。Agent 输出链接即可分享交互式架构图，无需本地安装任何工具。
