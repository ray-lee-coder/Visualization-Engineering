# tldraw

> 无限画布 SDK — 手绘风格 + 协作

| 属性 | 值 |
|------|-----|
| **仓库** | https://github.com/tldraw/tldraw |
| **类型** | React 画布 SDK |
| **安装** | `npm install tldraw` |

## 核心能力

- 无限缩放画布
- 手绘风格（SVG 路径）
- 实时协作（CRDT / Yjs）
- 形状 / 箭头 / 文本 / 图片 / 便签
- 可嵌入任意 React 应用

## Agent 场景

Agent 通过 tldraw JSON 格式生成结构化画布内容。配合 tldraw SDK 可实现 Agent 直出的手绘风格架构 / 白板 / 流程图。
