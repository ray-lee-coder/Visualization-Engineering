# React Flow / xyflow

> 节点式工作流 / 图编辑 UI 库

| 属性 | 值 |
|------|-----|
| **仓库** | https://github.com/xyflow/xyflow |
| **类型** | React / Svelte UI 库 |
| **安装** | `npm install @xyflow/react` |

## 核心能力

- 拖拽式节点编辑器
- 自定义节点 / 边样式
- 缩放 / 平移 / 键盘快捷键
- 迷你地图 / 控制面板
- 内置布局算法 (Dagre)

## Agent 场景

Agent 生成 JSON 节点数据 → React Flow 渲染交互式工作流图。适合 LLM 工作流编排、Pipeline 编辑器、知识图谱。
