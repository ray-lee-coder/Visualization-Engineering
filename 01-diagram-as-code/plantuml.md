## 定位
老牌文本转 UML 工具，专注软件工程建模。覆盖时序图、类图、部署图、组件图、用例图等 UML 全家桶，也支持非 UML 图（思维导图、JSON 图等）。Java 实现，需本地或服务器渲染。

## 输入 → 输出
`@startuml` / `@startuml` 包裹的 DSL 文本 → PNG/SVG 图表

## Agent 适配
较好。UML 语法成熟固定，LLM 对标准时序图和类图输出可靠。但需要额外安装 Java 运行环境或远程服务器，渲染链略长。

## 典型场景
- UML 序列图、类图的标准化建模
- 架构文档中的严格 UML 规范
- 持续集成中的自动化图表生成

## 一句话判断
UML 标准建模的行业事实标准，生态最成熟。
