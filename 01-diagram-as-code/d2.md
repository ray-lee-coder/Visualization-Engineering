## 定位
现代化图表脚本语言，专为可读性和生产力设计。编译式 CLI，内置 dagre/ELK/TALA 多种布局引擎，支持 `d2 watch` 实时预览。语言工具链完善：语法检查、格式化、导出。

## 输入 → 输出
`.d2` 源文件 → SVG/PNG/PDF 图表

## Agent 适配
较好。语法简洁清晰（接近自然语言），LLM 输出稳定。但编译期错误不够友好，复杂布局需调整参数，Agent 无法每次都一次成功。

## 典型场景
- 需要精确控制的架构图/拓扑图
- 自动化文档管线中的图表生成
- 追求美观默认布局的专业图表

## 一句话判断
比 Mermaid 更专业、比 Graphviz 更好写的现代 CLI 图表语言。
