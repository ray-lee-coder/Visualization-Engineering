## 定位
JavaScript 生态最主流的 Markdown 式图表工具。用类似 Markdown 的纯文本描述流程图/时序图/甘特图/类图/状态图/饼图/C4 图等，GitHub 原生渲染，npm 包可集成到任何前端项目。

## 输入 → 输出
Mermaid 代码块（\`\`\`mermaid） → SVG/PNG 图表

## Agent 适配
极高。纯文本 DSL，LLM 天生擅长生成和修正；语法直观，迭代成本低；GitHub Markdown 天然支持，结果可直接预览。Agent 直出效果最好的一档。

## 典型场景
- GitHub README / 文档内嵌图表
- 技术博客、设计文档中的流程图
- 快速绘制软件架构 C4 图
- 项目迁移或重构时的依赖关系可视化

## 一句话判断
所有免费 Markdown 文档需要插入图表的默认首选。
