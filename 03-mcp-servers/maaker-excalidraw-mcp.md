# maaker-excalidraw-mcp

**Repo:** maaker-ai/excalidraw-mcp

25+ diagram types via Excalidraw. CJK (Chinese/Japanese/Korean) text support for international diagrams. Sugiyama hierarchical layout algorithm for clean directed graph and flowchart rendering.

**Key features:** 25+ diagram types, CJK language support, Sugiyama layout engine, Excalidraw hand-drawn style.
