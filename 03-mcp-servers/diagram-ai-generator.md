# diagram-ai-generator

**Repo:** carlosmgv02/diagram-ai-generator

Multi-cloud architecture diagram MCP server. 5 tools covering AWS, Azure, GCP, and Kubernetes icon sets. Generates cloud-native architecture diagrams with provider-specific shape libraries.

**Key features:** Multi-cloud icons (AWS/Azure/GCP/K8s), 5 MCP tools, cloud architecture focus, provider-aware styling.
