# drawio-mcp-server

**Repo:** lgazo/drawio-mcp-server
**URL:** https://github.com/lgazo/drawio-mcp-server

Control draw.io directly via MCP. Includes a built-in editor for review and modification. Supports multi-document workflows — create, edit, and export multiple diagrams in a single session.

**Key features:** Built-in diagram editor, multi-document support, draw.io native format, MCP-controlled workflow.
