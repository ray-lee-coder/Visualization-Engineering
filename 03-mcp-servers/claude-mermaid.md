# claude-mermaid

**Repo:** veelenga/claude-mermaid
**URL:** https://github.com/veelenga/claude-mermaid

Mermaid preview server with live reload. Generates Mermaid diagrams and opens a browser preview that auto-updates on changes. Great for iterative diagram editing — tweak and see results instantly.

**Key features:** Live reload preview server, iterative editing workflow, browser-based rendering, instant visual feedback.
