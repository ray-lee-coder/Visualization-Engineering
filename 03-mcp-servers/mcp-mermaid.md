# mcp-mermaid

**Repo:** hustcc/mcp-mermaid
**URL:** https://github.com/hustcc/mcp-mermaid

Dynamic Mermaid diagram generation with built-in validation. Generates Mermaid.js code, validates syntax server-side before rendering, and returns working diagrams. Covers all standard Mermaid chart types.

**Key features:** Server-side Mermaid validation, all standard diagram types, syntax error detection, reliable rendering.
