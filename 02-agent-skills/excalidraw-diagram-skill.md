# excalidraw-diagram-skill

**Repo:** coleam00/excalidraw-diagram-skill
**URL:** https://github.com/coleam00/excalidraw-diagram-skill

Natural language → Excalidraw hand-drawn diagrams. Uses Playwright to render vector canvases with the signature sketch-style aesthetic. Great for wireframes, flowcharts, and informal architecture sketches.

**Key features:** Hand-drawn visual style, Playwright render engine, wireframe-friendly output, shareable .excalidraw files.
