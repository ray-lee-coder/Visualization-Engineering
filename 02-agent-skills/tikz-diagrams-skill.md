# tikz-diagrams-skill

**Repo:** Patrick-Healy/tikz-diagrams-skill
**URL:** https://github.com/Patrick-Healy/tikz-diagrams-skill

LaTeX TikZ diagram generation for academic and technical publications. Produces publication-quality vector graphics — block diagrams, graphs, automata, circuits, geometric figures. Compiles through LaTeX pipeline.

**Key features:** Academic-grade vector output, LaTeX-native, publication-ready graphics, extensive TikZ library support.
