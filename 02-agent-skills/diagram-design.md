# diagram-design

**Repo:** cathrynlavery/diagram-design
**URL:** https://github.com/cathrynlavery/diagram-design

14 editorial diagram types: comparison, timeline, flow, cycle, hierarchy, matrix, Venn, radial, pyramid, list, process, quadrant, table, network. Outputs HTML+SVG with brand color matching and WCAG AA compliance.

**Key features:** Editorial-grade visuals, brand-aware palette, accessibility-first, self-contained HTML output.
