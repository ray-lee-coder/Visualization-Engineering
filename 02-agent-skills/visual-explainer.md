# visual-explainer

**Repo:** nicobailon/visual-explainer
**URL:** https://github.com/nicobailon/visual-explainer

Rich HTML explainers and slide decks. 7 commands covering Mermaid diagrams, CSS Grid layouts, Chart.js visualizations, and interactive storytelling. Outputs standalone HTML presentations.

**Key features:** Multi-format slide decks, Chart.js + Mermaid integration, CSS Grid layout engine, interactive HTML output.
